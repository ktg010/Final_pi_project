Description

This project involves the raspberry pi and is mostly in python. The use of a camera, laser, and different type of glass makes up the additional items. The camera and laser is used to capture a picture of the light refactoring through different types of glass. This imaging is then processed to determine the refactor rate which is put through a dictionary to determine what type of glass was used.

information

The push for greener alternatives pushes for more recycling. This project is to be used to determine what pieces of glass can be reused to make glass through the refactor rate to determine what type of glass it is and if it is strong enough to be used.
